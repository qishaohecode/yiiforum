首页：

    2：最新的除了幻灯片的贴在外的10个帖子
    3：最新回复10条
    4：本周最热帖子10条，根据评论数排序
    5：查出板块及子板块及子版块最后回复的信息
    6：便民服务
    7：我要反馈
    8：登录、注册
    9：搜索功能
帖子列表页：
    1:板块列表
    2：帖子列表（帖子置顶+置顶排序+普通帖子按时间排序）
帖子详情页：
    1:读取帖子信息
    2：回复
